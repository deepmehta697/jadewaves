# Jade Waves SEO Playbook

## Goal

Rank for buyer-intent export queries, not only `from India` queries.

This means the site should target:
- product + supplier
- product + exporter
- product + bulk supplier
- product + application
- product + packing / specification / grade

`India` should stay in supporting copy and trust signals, but it should not be the only search angle.

## Search Console checklist

1. Verify the full domain in Google Search Console using a DNS TXT record.
2. Submit `https://jadewavesenterprise.com/sitemap.xml`.
3. Run URL Inspection on:
   - homepage
   - products page
   - top 5 priority product pages
4. Request indexing after major content updates, not after every small edit.
5. Check these reports weekly:
   - Performance
   - Indexing
   - Page experience / Core Web Vitals
   - Enhancements
6. Track performance by:
   - query
   - page
   - country
7. Watch for countries that already show impressions, then create stronger content for those markets.

## Content rules

- Every page should target one primary keyword cluster.
- Keep titles descriptive, not stuffed.
- Keep visible copy useful for real buyers: applications, specs, forms, packing, documents, destination handling.
- Add evidence that increases trust:
  - product photos
  - packing photos
  - dispatch photos / videos
  - clear specs
  - export document references
